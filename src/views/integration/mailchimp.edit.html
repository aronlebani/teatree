Set up Mailchimp integration

Steps

  1. Log into Mailchimp.
  2. Click your username, select Account, and navigate to the API keys screen.
  3. Hit Extras and select API Keys.
  4. Scroll down and select Create a Key.
  5. Copy the newly generated Key, and paste it into the field below.
Note that if you have multiple mailing lists, subscribers will automatically be subscribed to the first list.

*This is only used to enable the integration. We don't store your API key on our server.
